# エンドポイント

API サーバーの主要なエンドポイント一覧です。

## ゲーム進行管理

### 1. ゲーム進行状況の取得

- **URL**: `localhost:8000/api/game/progress`
- **メソッド**: `GET`
- **説明**: 現在のゲームの進行状況（おもちゃの総数と片付けられた数）を取得します。
- **レスポンス形式**:
  ```json
  {
    "total_toys": int,      // 登録されているおもちゃの総数
    "cleaned_toys": int     // 現在片付けられた（スキャンされた）おもちゃの数
  }
  ```

### 2. ゲームのリセット

- **URL**: `localhost:8000/api/game/reset`
- **メソッド**: `GET` / `POST`
- **説明**: ゲームの進行状況をリセットします。これまでのスキャン履歴と進行データがすべて削除され、新しいゲームを開始できる状態になります。
- **レスポンス形式**:
  ```json
  {
    "message": "Game progress reset successfully"
  }
  ```
